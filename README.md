# CSC242Project
# Week 4 Group Project for CSC242
# Group Members: Jhoana Mayle, Ever Hernandez, Eric Mounivong
# Description: This project is a word checker and spell checker for the English language. It uses a dictionary of words to check if a given word is spelled correctly or not.
# The project is implemented in CPP and uses a hash table to store the dictionary of words. The hash table is implemented using separate chaining to handle collisions. 
# The project also includes a function to suggest similar words if the given word is not found in the dictionary.
